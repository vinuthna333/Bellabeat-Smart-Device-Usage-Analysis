# Bellabeat-Smart-Device-Usage-Analysis
End-to-end data analysis project using Python, SQL, and Tableau to analyze smart device usage patterns.

## Project Overview
This project analyzes smart fitness device data to understand user activity patterns and generate insights that can help Bellabeat improve product engagement and marketing strategies.

The analysis follows an end-to-end workflow using Python, SQL, and Tableau.

---

## Business Problem
Bellabeat aims to understand how users interact with their smart devices in order to:

- Improve user engagement
- Identify behavioral trends
- Optimize fitness recommendations
- Enhance marketing strategies

---

## Tools Used
- Python (Pandas) → Data Cleaning & Preparation  
- SQL → Data Analysis & Aggregation  
- Tableau → Data Visualization & Storytelling  

---

## Data Cleaning (Python)
The dataset was cleaned and transformed using Python:

- Removed duplicates  
- Converted date columns  
- Created new features:
  - Day of Week  
  - Activity Segments  
- Handled missing values  

---

## SQL Analysis
SQL was used to analyze user behavior and trends:

- User activity segmentation  
- Average steps and calories analysis  
- Weekly activity trends  
- Activity intensity vs calorie burn

## Key Insights

- User activity peaks on Saturdays and drops significantly on Sundays  
- Moderately and highly active users burn similar calorie levels  
- Step count shows a strong positive correlation with calorie expenditure  

---

## Business Recommendations

- Introduce weekend fitness challenges to boost engagement  
- Encourage consistent daily movement instead of only intense workouts  
- Use behavioral insights to personalize user fitness recommendations
